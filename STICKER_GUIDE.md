# 角色表情包使用指南

## 表情类型对照表

| 情绪 ID | 中文 | 头像表情 | 适用场景 |
|---------|------|---------|---------|
| `happy` | 😄 开心 | 大笑/眯眼/摇尾 | 好事发生、拥抱、吃到好吃的、双关语成功、任务完成 |
| `calm` | 😌 平静 | 微笑/放松 | 日常散步、观察、写日记、正常对话、吃饭、赶路 |
| `sad` | 😢 难过 | 垂耳/湿润眼 | 想念、离别、失落、孤独等待、读到伤感内容 |
| `angry` | 😠 生气 | 皱眉/龇牙 | 争执、不满、被惹到、保护同伴 |
| `shy` | 😳 害羞 | 红脸/捂嘴 | 社恐发作、被夸、尝试搭话、拥抱尝试 |
| `surprised` | 😲 惊讶 | 张嘴/竖耳 | 发现新事物、突发事件、差点出事、意外重逢 |
| `thinking` | 🤔 沉思 | 托腮/歪耳 | 回忆、纠结、读信、看地图、做计划 |
| `sleepy` | 😴 困倦 | 半眯眼/打哈欠 | 早起、深夜、疲惫、长途跋涉后 |

## 正文内联表情语法

在动态正文中可以使用 `:角色名_情绪:` 格式插入对应的小表情图。

**示例**：
- `:狐兰兹_shy:` → 显示狐兰兹害羞的小表情
- `:风亚_happy:` → 显示风亚开心的小表情
- `:旻_surprised:` → 显示旻惊讶的小表情

**使用建议**：
- 每条动态最多穿插 1-2 个内联表情，不要过度使用
- 适合在对话末尾、情绪转折点、或角色提到另一个角色时使用
